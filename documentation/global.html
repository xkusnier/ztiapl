GameLoop()

The game loop function that continuously updates and renders the game while the game is running. If the game is not paused, the function calls updateAll(), drawAll(), CheckGameOver(), and CheckAbleToShoot() functions, and increments the frame counter. If the game is not over and is not paused, it requests the next animation frame using requestAnimationFrame() function to continue the game loop. If the game is over, the function calls RestartGame() function to restart the game.
